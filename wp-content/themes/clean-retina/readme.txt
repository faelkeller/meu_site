== Description ==

Clean Retina is a simple, clean and responsive WordPress theme which is compatible to popular plugins like WooCommerce, bbPress and many more.

== Installation ==

= Manual installation: =

1. Upload the `clean-retina` folder to the `/wp-content/themes/` directory

= Activiation and Use =

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Customize to change theme specific options

== Frequently Asked Questions ==

= How to use featured Post Slider? =

* Create Post, Add featured image to the Post.
* Add all the Post ID that you want to use in the featured slider.
* Featured Slider will show featured images, Title and excerpt of the respected added post's IDs.
